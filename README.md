# 文件管理系统简介 java864

## 项目概述

本项目是一个基于Spring Boot和Vue的前后端分离的文件管理系统。它功能齐全，实用性强，能够满足不同用户对文件管理的高效和便捷需求。

## 技术栈

- 后端：Spring Boot + MyBatis + MySQL
- 文件存储：Go-fastdfs
- 前端：Vue + Element UI

## 功能模块

### 系统角色

系统定义了以下两种角色：

- 管理员：可以查看所有用户的文件，管理用户和系统设置。
- 普通用户：只能查看和操作自己的文件。

### 主要功能

- 角色管理与权限管理：灵活配置用户角色和权限。
- 文件操作：
  - 文件分享：用户可以对文件进行分享。
  - 文件删除：用户可以删除不再需要的文件。
  - 文件详情：用户可以查看文件的详细信息。

## 系统运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：现代浏览器，建议使用Chrome或Firefox

## 部署步骤

1. 部署后端服务（Spring Boot应用）。
2. 部署数据库（MySQL）。
3. 部署文件存储系统（Go-fastdfs）。
4. 部署前端服务（Vue应用）。

## 目录结构

```
根目录/
├── backend               # 后端代码目录
├── frontend              # 前端代码目录
└── storage               # 文件存储目录
```

## 核心亮点

- **前后端分离**：系统采用前后端分离的架构，有利于前后端分工合作，提高开发效率。
- **灵活的用户角色与权限管理**：系统可以根据不同用户的职责赋予不同的角色和权限。
- **可扩展的文件存储方案**：基于Go-fastdfs，具有良好的可扩展性和稳定性。
